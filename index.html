Regras do Jogo

  1. O objetivo é guiar o sapo até o outro lado do rio, evitando os obstáculos.
  2. Use as setas do teclado (↑, ↓, ←, →) para mover o sapo.
  3. Evite os carros que passam pela estrada - eles podem esmagar o sapo!
  4. No rio, pule em cima dos troncos para atravessar com segurança.
  5. Cuidado com os troncos que podem levar o sapo para fora da tela!
  6. Complete todas as áreas de chegada para vencer o jogo.
